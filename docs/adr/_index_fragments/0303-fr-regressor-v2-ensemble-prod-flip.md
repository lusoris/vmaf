Adds `ai/scripts/train_fr_regressor_v2_ensemble_loso.py` — a 9-fold leave-one-source-out trainer over the Netflix Public Dataset for the five seeds, emitting `loso_seed{N}.json` artefacts per seed — and `scripts/ci/ensemble_prod_gate.py` — the production-flip gate that returns exit 0 iff `mean_i(PLCC_i) >= 0.95` AND `max_i(PLCC_i) - min_i(PLCC_i) <= 0.005`. The variance bound is load-bearing: it protects the predictive-distribution semantics that the in-flight `vmaf-tune --quality-confidence` flag (ADR-0237 consumer) relies on; without it, the ensemble mean could mask a one-seed-wins-four-seeds-tie configuration that breaks the conformal calibration assumption. Per-seed rows flip `smoke: true -> false` only after that seed clears its individual `PLCC_i >= 0.95` gate; the ensemble-mean entry (if/when registered) flips only after all five seeds clear. Workflow wiring is intentionally deferred to the follow-up flip PR (no real `loso_seed{N}.json` artefacts on master to gate on yet).
