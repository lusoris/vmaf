Ships `tools/vmaf-tune/src/vmaftune/codec_adapters/{h264_amf,hevc_amf,av1_amf,_amf_common}.py` plus `tools/vmaf-tune/tests/test_codec_adapter_amf.py`. Compresses the canonical 7-level preset vocabulary onto AMF's 3 quality rungs (`placebo`/`slowest`/`slower`/`slow` → `quality`; `medium` → `balanced`; `fast`/`faster`/`veryfast`/`superfast`/`ultrafast` → `speed`) — AMF hardware does not expose finer steps. Rate control is `-rc cqp` with matched `-qp_i` / `-qp_p` (closest analogue to x264 CRF); QP range pinned to the (15, 40) Phase A informative window for cross-codec comparability. AV1 is RDNA3+ only (Radeon RX 7000 series and newer). `ensure_amf_available` probes `ffmpeg -encoders` for the requested AMF codec and rejects builds without `--enable-amf` or AMD GPU.
